## Даалгавар

### Бодлого 1: Hangman

Эхний даалгавраар Hangman хэмээх үгэн тоглоомын хялбар хувилбарыг хөгжүүлнэ. Хэрэв та энэхүү тоглоомыг мэдэхгүй бол дараах холбоосноос дэлгэрэнгүй үзээрэй. [https://en.wikipedia.org/wiki/Hangman\_(game)](<https://en.wikipedia.org/wiki/Hangman_(game)>).

Бидний энэ бодлого нь дээрх зааврыг бүрэн хуулаагүй бөгөөд хялбарчилсан болон шаардлагатай функцүүдийн үүсгэх файлыг хавсаргаж өгсөн болно.

#### A) Эхлэл

`hangman.py` болон `words.txt` файлуудыг даалгаварын заавартай хамт хавсаргасан бөгөөд нэг хавтаст хадгалах шаардлагатай. hangman.py файлыг кодоо бичиж эхлэхээсээ өмнө ажиллуулж зөв хавтаст хадгалсан эсэхээ шаалгаарай. Учир нь words.txt файлтай хамт нэг хавтаст хадгалсан бол кодыг ажиллуулахад доорх үр дүнг харуулж байх ёстой.

```text
>>>
Loading word list from file...
  55900 words loaded.
>>>
```

#### B) Hangman тоглоомын шаардлага

`hangman` гэсэн функцийн кодыг бичих бөгөөд компьютертэй тоглоно. Компьютер нь нэг үг сонгох бөгөөд тоглогч тухайн үгэнд орсон үсгийг таах замаар тоглоно.

1. Компьютер нь `words.txt` файлаас нэг үгийг санамсаргүй байдлаар сонгоно. Тус файлд бүх үгийг жижиг үсгээр бичсэн байгаа болно.
2. Тоглогч эхлээд тодорхой нэг тооны санамсаргүй утга оруулна. Энэ тоо нь тоглогч хэдэн удаа үсэг таах тоо юм.
3. Тоглоом давталт байдлаар үргэлжилнэ. Ингэхдээ тоглогч үсэг оруулах ба компьютер доорх хоёр хариуны аль нэгийг өгнө:
   1. Хэрэв сонгосон үгэнд тоглогчийн оруулсан үсэг байвал харуулна.
   2. Байхгүй бол шийтгэл өгөх ба үлдсэн таах боломжтой тоог харуулна.
4. Тоглоом нь тоглогч бүх үсгийг тааж дуусахад эсвэл таах тоо нь дуусахад зогсоно.

### Бодлого 2: 1 дүгээр хэсэг - 3 туслах функц

Тоглоомын кодыг бичиж эхлэхээсээ өмнө бид бодлогоо логик дэд хэсгүүдэд задалж, энэ тоглоомыг ажиллуулахад шаардлагатай гурван туслах функцийг бүтээх болно. Энэ бол ямар нэг асуудлыг тооцооллын аргаар шийдвэрлэхэд хэрэглэдэг түгээмэл арга бөгөөд бид энэ аргыг хэрэглэж хэвших хэрэгтэй.

`hangman.py` файлд тоглоомд хэрэглэгдэж болох хэд хэдэн функцийг хэрэгжүүлсэн байгаа. Файлын дээд талд байгаа хоёр функцийн кодыг тоохгүй байж болно, гэхдээ та тэдгээрийн тайлбар буюу docstring-г (функц бүрийн эхэнд байрлах `"""` дотох бичвэр) уншиж функц бүрийг хэрхэн ашиглах талаар ойлгох хэрэгтэй.

#### A) Үгийг таасан эсэхийг шалгах

Эхлээд, `is_word_guessed()` функцийн кодыг бичнэ. Энэ функц нь хоёр string параметр авдаг ба `secret_word` болон `letter_guessed` гэсэн нэртэй параметрүүд байна. Энэ функц нь `secret_word`-ийн бүх үсгийг таасан бол Boolean утга `True` буцаах ба эсрэг тохиолдолд `False` утга буцаана. Энэ функц нь тоглоом амжилттай дууссан эсэхийг тодорхойлох бөгөөд үсэг бүрийг шалгах давталт зогсох үндсэн нөхцөл болох юм.

Энэ функцийн авч байгаа параметрийн утгууд нь бүгд жижиг үсгээр бичсэн байна гэж тооцоорой.

**Ажиллаж буй жишээ:**

```python
>>> secret_word = 'apple'
>>> letters_guessed = ['e', 'i', 'k', 'p', 'r', 's']
>>> print(is_word_guessed(secret_word, letters_guessed))
False
```

#### B) Тоглогчийн таамаглалыг авах

Энэ удаа `get_guessed_word()` функцийн кодыг хөгжүүлэх бөгөөд мөн хоёр string төрлийн `secret_word` болон `letters_guessed` парамертүүд авдаг байна. Энэ функц нь нийлмэл үсгүүд болон доогуур зураас буцаах бөгөөд `secret_word` дэх аль аль үсгүүд `letters_guessed` дотор байгаа гэдгээс хамаарч буцаах юм. Энэ нь өмнөх `is_word_guessed()` функцээс тийм ч их ялгаатай биш юм.

Бид доогуур зураасны араас хоосон зайтай `(_ )` хамт хэрэглэх бөгөөд үүгээр үл мэдэгдэх үсгийг илэрхийлнэ. Өөр тэмдэгт сонгож болох ч тоглоомд илүү тохирох дүрс гэж үзсэн болно. Тэмдэглэхэд хоосон зай бол маш чухал бөгөөд учир нь доогуур зурааснуудыг хооронд нь ялгахад хүндрэлтэй байдал үүснэ. Энийг программын хэрэглэхэд ойлгомжтой байдал гэдэг бөгөөд таны бичсэн программыг хэрэглэхэд ойлгомжтой байх нь чухал юм. Хэрэв хэрэглэгч таны бичсэн программ хэрэглэхэд болон ажиллуулахад хүндрэлтэй байвал хэрэглэхгүй учир та программ хөгжүүлж байхдаа энэ тухай анхааралтай бодож байх ёстой юм.

**Тусламж:** Функцийн зохиомжийг гаргах үедээ функц ажиллаад дуусахад ямар мэдээлэл буцаах хэрэгтэй, давталт хийх үед тухайн мэдээллээ хадгалах газар хэрэгтэй эсэх болон бодолт хийсэн үр дүн дээрээ хэрхэн мэдээлэл нэмэх тухай бодож үзээрэй.

**Ажиллаж буй жишээ:**

```python
>>> secret_word = 'apple'
>>> letters_guessed = ['e', 'i', 'k', 'p', 'r', 's']
>>> print(get_guessed_word(secret_word, letters_guessed))
'_ pp_ e'
```

#### C) Ашиглаж болох үсгүүдийг авах

Энэ удаа, `get_available_letters()` функцийг хэрэгжүүлэх бөгөөд нэг жагсаалт төрлийн параметр буюу `letters_guessed` хувьсагчийг авдаг байна. Энэ функц нь жижиг латин үсгүүд буцаах ба `letters_guessed`-д байгаа үсгүүдийг буцаахгүй. Өөрөөр хэлбэл тоглогчийн таагаагүй үсгүүдийг л буцаана.

Энэ функц нь цагаан толгойн дарааллаар үсгүүдийг буцаана. Энэ функцэд ашиглагдаж байгаа `letters_guessed` утгууд нь мөн бүгд жижиг хэмжээтэй үсэгтэй байна гэж тооцоорой.

**Тусламж:** `string.ascii_lowercase` функцийг ашиглаж бүх жижиг латин үсгүүдийг авч болно.

```python
>>> import string
>>> print(string.ascii_lowercase)
abcdefghijklmnopqrstuvwxyz
```

**Ажиллаж буй жишээ:**

```python
>>> letters_guessed = ['e', 'i', 'k', 'p', 'r', 's']
>>> print get_available_letters(letters_guessed)
abcdfghjlmnoqtuvwxyz
```

### Бодлого 3: 2 дугаар хэсэг - Тоглоом

Та өөрт хэрэг болох функцүүдтэй болсон бөгөөд одоо `hangman` функцийг хөгжүүлэх болно. Энэ функц нь `secret_word` гэсэн ганц параметр авна. Эхлээд та гараас тэр үгийг авдаг байдлаар хийж үзэх нь туршихад хялбар байх болно. Дараа нь компьютер санамсаргүйгээр файлаас авч тантай тоглодог болгох юм.

`hangman` функцийг дуудсанаар тоглогч компьютер хоёр тоглож эхлэх юм. Кодоо бичихдээ туслах гурван функцийг (`is_word_guessed`, `get_guessed_word`, `get_available_letters`) ашиглаж бичихээ бүү мартаарай.

Доорх тоглоомын шаардлага нь хэд хэдэн ангилалд хуваагдана. Тиймээс бичсэн программ тэр бүх шаардлагыг хангаж байгаа эсэхийг шалгаарай.

#### Тоглоомын шаардлага

**A. Тоглоомын архитектур:**

1. Компьютер нь `words.txt` файлд байгаа үгнүүдээс нэг үгийг санамсаргүй сонгодог байна. Энэ үйлдлийн бид аль хэдийн `hangman.py` файлд файлаас уншиж үгийн жагсаалтыг аван санамсаргүй үг сонгож өгөх функцүүдийг хөгжүүлсэн байгаа.
2. Тоглогч 6 үсэг таах боломжтой (Энэ нь зөвхөн буруу таасан үед хорогдох ба тоглоомын дүрэм хэсгээс дэлгэрэнгүй хараарай).
3. Тоглоом эхлэхэд, компьютер нь тоглогчид таах үг нь хэдэн үсэгтэй, мөн хэдэн удаа үсэг таах боломжтойг мэдэгдэнэ.
4. Компьютер нь тоглогчийн таамаглаж хэлээгүй бүх үсгийг бүртгэж явах бөгөөд тоглогч үсэг хэлэх бүр үлдсэн хэлээгүй үсгүүдийг харуулдаг байна.

Тоглоомыг хэрэгжүүлсэн жишээ:

```text
Loading word list from file...
55900 words loaded.
Welcome to the game Hangman!
I am thinking of a word that is 4 letters long.
-------------
You have 6 guesses left.
Available letters: abcdefghijklmnopqrstuvwxyz
```

**B. Тоглогч-Компьютер хоёрын харилцаа:**

Тоглоом нь доорх дарааллаар харилцдаг байна:

1. Үсэг таах бүрийн өмнө тоглогчдод дараах мэдээллийг харуулдаг байна:
   1. Хэдэн үсэг таахаар үлдсэн тоог сануулдаг байна.
   2. Тоглогчийн хэлээгүй бүх үсгүүдийг харуулна.
2. Тоглогч үсгийг нэг нэгээр нь тааж явна. (Хэрэглэгчийн оролтын шаардлагыг доод хэсгээс хараарай).
3. Үсэг таамаглах бүрд таалгаж байгаа үгэнд байгаа эсэхийг тоглогчид мэдэгдэнэ. Ингэхдээ таагаагүй үсгүүд нь доогуур зураас, хоосон зайгаар `(_ )` харагдах бол таасан үсэг нь тухайн үгнийхээ зөв байрлалд харагдана.
4. Таамаглаж дууссаны дараа дараагийн таамаглах үйлдлийн хооронд `(----)` дундуур зураасаар зааглаж харуулна.

Тоглоомыг хэрэгжүүлсэн жишээ:
(Цэнхэр үсэг нь тоглогч оруулсан үсэг бол бусад текстүүд нь компьютерээс дэлгэц дээр гаргасан байдал юм)

```text
You have 6 guesses left.
Available letters: abcdefghijklmnopqrstuvwxyz
Please guess a letter: a
Good guess: _ a_ _
------------
You have 6 guesses left.
Available letters: bcdefghijklmnopqrstuvwxyz
Please guess a letter: b
Oops! That letter is not in my word: _ a_ _
```

**C. Хэрэглэгчийн оролтын шаардлага:**

1. Хэрэглэгч нь нэг удаадаа ганц тэмдэгт оруулах ба тусгай тэмдэгт, тоо, том жижиг үсэг оруулж болно!
2. Хэрэв хэрэглэгч үсгээс өөр утга оруулбал (тусгай тэмдэгт) хэрэглэгчид зөвхөн үсэг оруулах ёстой болохыг мэдэгдэнэ. Учир нь хэрэглэгч санамсаргүй өөр тэмдэгт оруулж болох юм. Хэрэглэгч 3 удаа ийм сануулга авах боломжтой, мөн таасан үсгээ бас оруулах юм бол сануулга өгнө. Тоглогчийн сануулга авах боломж дуусмагц таах боломжийг нэгээр хорогдуулна.

**Тусламж #1:** `input` функцийг дуудаж хэрэглэгчийн тааж буй утгыг авна.

1. Хэрэглэгчийн оролт цагаан толгой үсэг эсэхийг шалгана.
2. Хэрэв хэрэглэгч том жижиг үсэг оруулахгүй бол сануулга өгч, сануулга авах боломжийн тоог нэгээр багасгана.

**Тусламж #2:** Тэмдэгтийн цуваатай ажиллах `str.isalpha('your string')` болон `str.lower('Your String')` функцүүд хэрэг болох байх! Хэрэв энэ ямар учиртай функц болохыг мэдэхгүй бол `help(str.isalpha)` эсвэл `help(str.lower)` гэж Spyder shell дээр бичиж ажиллуулан функцийн тухай мэдээллийг авах боломжтой.

**Тусламж #3:** `words.txt` файл дахь бүх үг жижиг үсгээр бичигдсэн учир хэрэглэгчийн оролтыг зөвхөн жижиг үсгээр болгох хувиргах шаардлагатай байх. Тоглоом нь жижиг үсэгтэй ажиллах ёстой.

Тоглоомыг хэрэгжүүлсэн жишээ:

```text
You have 3 warnings left.
You have 6 guesses left.
Available letters: bcdefghijklmnopqrstuvwxyz
Please guess a letter: s
Oops! That letter is not in my word: _ a_ _
------------
You have 5 guesses left.
Available letters: bcdefghijklmnopqrtuvwxyz
Please guess a letter: $
Oops! That is not a valid letter. You have 2 warnings left: _ a_ _
```

**D. Тоглоомын дүрэм:**

1. Тоглогч 3 сануулгын эрхтэйгээр тоглоом эхэлнэ.
2. Хэрэв тоглогч үсгээс өөр тэмдэгт оруулбал тоглогчид зөвхөн үсэг оруулах ёстой болохыг мэдэгдэнэ.
   1. Хэрэв тоглогч нэг эсвэл түүнээс олон сануулга авах боломжтой бол сануулгын боломжийн тоог нэгээр багасгаж тоглогчид хэдэн сануулга авах боломжтой үлдсэнийг мэдэгдэнэ.
   2. Тоглогчийн сануулга авах боломж дууссан бол таамаглах тоог нэгээр хорогдуулна.
3. Хэрэв тоглогчийн оруулсан үсэг өмнө таамагласан үсэг бол тоглогчид өмнө таамагласан үсэг болохыг мэдэгдэнэ.
   1. Хэрэв тоглогч нэг эсвэл түүнээс олон сануулга авах боломжтой бол сануулгын боломжийн тоог нэгээр багасгаж тоглогчид хэдэн сануулга авах боломжтой үлдсэнийг мэдэгдэнэ.
   2. Тоглогчийн сануулга авах боломж дууссан бол таамаглах тоог нэгээр хорогдуулна.
4. Хэрэв тоглогчийн оруулсан үсэг өмнө таамаглаагүй үсэг бөгөөд таалгах гэж буй үгэнд байгаа бол тоглогчийн таамаглах тоог багасгахгүй.
5. **Гийгүүлэгч:** Хэрэв тоглогчийн таамаглаж оруулсан үсэг гийгүүлэгч бөгөөд гийгүүлэгч нь таамаглах ёстой үгэнд байхгүй бол таамаглах тоо нэгээр багасна.
6. **Эгшиг:** Хэрэв тоглогчийн таамаглаж оруулсан үсэг эгшиг бөгөөд таамаглах ёстой үгэнд байхгүй бол таамаглах тоо хоёроор багасна.

Хэрэгжүүлсэн жишээ:

```text
You have 5 guesses left.
Available letters: bcdefghijklmnopqrtuvwxyz
Please guess a letter: t
Good guess: ta_ t
------------
You have 5 guesses left.
Available letters: bcdefghijklmnopqrtuvwxyz
Please guess a letter: e
Oops! That letter is not in my word: ta_ t
------------
You have 3 guesses left.
Available letters: bcdfghijklmnopqrtuvwxyz
Please guess a letter: e
Oops! You've already guessed that letter. You now have 2 warnings:
ta_ t
```

**E. Тоглоом дуусах:**

1. Тоглоом нь тоглогч бүх үсгийг таасан эсвэл таамаглах тоо дуусахад тоглоом дуусна.
2. Хэрэв тоглогч бүх үсгийг тааж амжаагүй бөгөөд таамаглах тоо дууссан бол тоглогчид тоглоомд хожигдсон болохыг мэдэгдэж, таах ёстой үгийг бүрэн харуулна.
3. Хэрэв тоглогч хожсон бол баяр хүргэсэн мессеж хэвлэж, тоглогчид түүний оноог харуулна.

### Бодлого 4: 3 дугаар хэсэг - Тусламж авч тоглох

Компьютертой тоглох үед компьютерыг хожих нь тийм ч амархан биш болохыг мэдэх болно. Ялангуяа олон хүн мэддэггүй esoteric үг сонгогдсон бол. Тиймээс компьютероос тусламж хүсэж болох юм. Тухайлбал тоглогчийн таасан үсгүүд орсон үгийн жагсаалт харах гэх мэт.

Жишээ нь нууц үг нь "tact" бөгөөд тоглогч "t" үсгийг таагаад байх үед тоглогч таах үг нь одоогоор ийм "t\_ \_ t" байна гэж мэдэх боловч бусад үсгийг таах хэрэгтэй. Тэгвэл доорх байдлаар харуулдаг бол илүү таахад амар байх боломжтой:

```text
tact tart taut teat tent test text that tilt tint toot tort tout trot tuft twit
```

Бид одоо Hangman тоглоомын өөр нэг хувилбар болох (`hangman_with_hints` гэсэн нэртэй) компьютероос `*` тусгай тэмдэгтээр тусламж авах боломжтой тоглоомын горим хөгжүүлэх болно. Энэхүү `*` тусгай тэмдэгтийг тоглогч оруулахад компьютер нь файл дээрх нийт үгээс одоогоор тоглогчийн таасан байгаа үсэг, үсгийн байрлал тохирох үгсийн жагсаалт харуулдаг байх юм. Мэдээж тоглоомын эхэнд од тусгай тэмдэгт оруулах юм бол 55,900 үг буюу файлын бүх үг гарч ирэх тул тоглоомын эхэнд хэрэглэхгүй байхыг зөвлөж байна. Харин бидний таамаглах тоо дуусаж байгаа үед болон хэд хэдэн үсгээ таачихсан үед хэрэглэвэл илүү үр дүнтэй юм.

Ингэхийн тулд эхлээд дараах хоёр туслах функцийг хэрэгжүүлэх хэрэгтэй:

#### A) Тааж байгаа үгтэй тааруулах

`match_with_gaps` функц нь хоёр параметр авна: `my_word` болон `other_word`. `my_word` таамаглах гэж байгаа үгийн тухайн агшинд ямар байгаа байдлыг харуулсан хэлбэртэй үг байна. Өөрөө хэлбэл `_` буюу доогуур зураас орсон (жишээлбэл "t\_ \_ t" гэсэн хэлбэртэй) үг байх юм. Харин `other_word` нь зөв бичлэгтэй англи үг байна.

Энэ функц нь `my_word` болон `other_word` үгсийн доогуур зурааснаас бусад харгалзах байрлал дээрх үсгүүд нь ижил бол `True` гэсэн утга буцаах бөгөөд үгийн урт өөр харгалзах байрлал дах үсгүүд ижил биш бол `False` гэсэн утга буцаадаг байна.

Үсэг таасан үед тааж байгаа үгт тухайн үсэг байх бүх байрлал дээр гарч ирэх бөгөөд доогуур зураасыг (`_`) үсгээрээ солих ёстой гэдгийг санаарай.

Ажиллаж байгаа жишээ:

```python
>>> match_with_gaps("te_ t", "tact")
False
>>> match_with_gaps("a_ _ le", "banana")
False
>>> match_with_gaps("a_ _ le", "apple")
True
>>> match_with_gaps("a_ ple", "apple")
False
```

**Тусламж:** Үгийн уртыг харьцуулахдаа хоосон зайг хасах ёстой учир хоосон зай хасах `strip()` функцийг хэрэглэж болно.

#### B) Бүх таарах боломжийг харуулах

`show_possible_matches` функц нь нэг параметр авна: `my_word` буюу таамаглах гэж байгаа үгийн тухайн агшинд ямар байгаа байдлыг харуулсан хэлбэртэй үг байна. Өөрөөр хэлбэл `_` буюу доогуур зураас орсон (жишээлбэл "t\_ \_ t" гэсэн хэлбэртэй) үг байх юм.

Энэ функц `wordlist` (файлын эхэнд тодорхойлсон байсан)-д байгаа бүх үгээс `my_word`-той тохирох бүх үгийг дэлгэцэд хэвлэдэг байна. Хэрэв тохирох үг байхгүй бол "тохирох үг олдсонгүй" гэж дэлгэцэд хэвлэнэ.

Ажиллаж байгаа жишээ:

```python
>>> show_possible_matches("t_ _ t")
tact tart taut teat tent test text that tilt tint toot tort tout trot tuft twit

>>> show_possible_matches("abbbb_ ")
No matches found

>>> show_possible_matches("a_ pl_ ")
ample amply
```

#### C) Тусламж авдаг Hangman тоглоом

Одоо бид `hangman` тоглоомынхоо кодоос `hangman_with_hints` хувилбарыг үүсгэх боломжтой болсон. Од тэмдэгт буюу `*`-г тоглогч оруулах үед бүх боломжит үгийг дэлгэцэд хэвлэн харуулах юм.

Хэрэв оруулсан тэмдэгт нь `*` бол тоглогчийн таах боломжийн тооноос хасахгүй байх ёстой.

Бидний өмнөх Hangman тоглоомын код дотор доорх хоёр мөрийг коммент буюу тайлбар мөр болгож:

```python
#secret_word = choose_word(wordlist)
#hangman(secret_word)
```

Харин шинэ хувилбар болох тусламжтай hangman тоглоомоо доорх хоёр мөр кодын коммент буюу тайлбар тэмдгийг авч хэрэглэнэ.

```python
#secret_word = choose_word(wordlist)
#hangman_with_hints(secret_word)
```
